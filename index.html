ABOUT

Hi! I'm Katarzyna. A beginner front-end developer with a strong need to create beautiful and useful websites/apps. Self taught geek who's always looking to learn and use new technologies. Reliable and open-minded team-player, flexible in work environment.

I wrote my first lines of code when I was about 11 and needed a cool theme for a blog. It was a silly Paint-made theme with code written in NotePad, but I will never forget that feeling of pride and joy when I finished it and it worked!

Who would've think, that thirteen years later this would become my passion and a career.
